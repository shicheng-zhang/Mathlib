# v11S Release Notes

## Theme

v11S is the transition from experimental development into a maintained release line.

## Included

- hardened core numerical primitives
- expanded edge-case validation
- FFT and DSP stabilization
- linear algebra safety checks
- closure validation tooling
- documented numerical contracts

## Deferred to v12A1

The following are intentionally not part of v11S:
- aggressive approximation redesigns
- next-generation SIMD architecture
- expanded arbitrary precision paths
- major API restructuring
